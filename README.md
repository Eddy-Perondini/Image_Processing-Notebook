# Image_Processing-Notebook

This github was created solely for image processing practices and study. 

All activities included in this image processing notebook, so far:

- Image equalization; 
- Histogram calculation;
- Noise addition;
- Clusterization process;
- Binarization;
- Quantitative metrics to evaluate images;
- Noise filtering.

Future proceedings: 

- Object detection;
- Object counting;
- Segmantation;
- Edge erosion.

etc.
